# regular-briefings

Daily AI Intelligence Brief, archive and radar portal.

- `index.html` portal hub: stats, hottest topics, top stories, searchable archive, dark/light.
- `briefs/YYYY-MM-DD.html` individual daily briefs (with shared topbar, theme toggle, jump nav, prev/next).
- `assets/brief-enhance.js` shared enhancement injected into every brief.
- `data/briefs.json` parsed data index powering the portal.

Deployed via Cloudflare Pages with Cloudflare Access (email one-time PIN).
