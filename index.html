arxiv 2605.14498 GroupMemBench benchmarks LLM agent memory in multi-party conversations

New benchmark for memory in group and channel settings versus the single-user dyadic assumption baked into MemBench, MemoryAgentBench and others. Tests three properties: group dynamics beyond concatenated one-on-one chats, speaker-grounded belief tracking and audience-adapted language (Theory-of-Mind shifts). Uses a graph-grounded synthesis pipeline with per-user personas and six adversarial query categories (multi-hop, knowledge update, term ambiguity, user-implicit reasoning, temporal, abstention). Source: arxiv.org / 2605.14498
