GitHub Copilot moves all monthly plans to usage-based "AI Credits" billing, effective today, June 1

Tier 1 High confidence Early signal Coding Tools Pricing

As of June 1, Copilot replaces premium request units with GitHub AI Credits, metered on input, output, and cached tokens at each model's published API rate (1 credit = $0.01). Monthly Pro and Pro+ users auto-migrate today; Pro+ stays $39 and now includes $39 of credits, Business $19, Enterprise $39, with Business and Enterprise getting promotional included usage through August. Code completions and next-edit suggestions stay unlimited and unbilled. Annual plan holders keep request-based pricing until renewal. Developer reaction on May 30-31 was sharply negative on cost predictability for long agentic sessions.

Source: GitHub Blog / TechCrunch, effective June 1, 2026

Why this matters before others see it: This is the largest AI coding subscription converting flat-rate seats into metered consumption, and it is the first real test of whether developers accept token billing for everyday tooling. The second-order effect is that long-running agentic loops become cost-unpredictable, which pushes teams to cap agent autonomy, watch token budgets per task, or self-host. It also hands a pricing wedge to rivals: Cursor, Windsurf, and Claude Code can now position on predictable or flat pricing. Watch "tokens per task" become a budgeting metric, and watch GitHub probe how far it can move price before churn shows up, because the May 30-31 backlash signals real elasticity.

Correction and signal: DeepSeek's 75% V4-Pro price cut is permanent, not an expiring promo, the list price did not revert

Tier 1 High confidence Early signal Competitor Pricing

This brief reported on May 31 that DeepSeek's V4-Pro promotional discount would expire that day and list pricing would return. That was a misread. DeepSeek announced on May 22 that the 75% discount becomes the permanent list price: roughly $0.435 per million input tokens, $0.87 per million output, and $0.003625 per million cached. The cut did not roll back on May 31. By widely cited comparisons, V4-Pro now sits below OpenAI, Anthropic, and Google on token cost, on the order of 11x cheaper than GPT-5.5 on input.

Source: Engadget / InfoWorld, announced May 22, 2026

Why this matters before others see it: Two things. First, the correction: a scheduled "promo ends" framing circulated across multiple outlets and into this brief, when the real move was the opposite, a permanent reset. Worth flagging because the wrong version is still propagating. Second, the signal: a frontier-tier model holding a permanent sub-$0.50 input price sets a hard floor that squeezes everyone selling inference above it. Pair this with the Copilot item and the same week shows AI cost being repriced from both ends, the model layer and the tooling layer. The pressure lands hardest on mid-tier API resellers and on any product whose margin depends on inference staying expensive.

Gemini 2.0 Flash and 2.0 Flash-Lite shut down today, June 1, old model names start returning errors

Tier 1 High confidence Competitor Lifecycle

Google retires gemini-2.0-flash and gemini-2.0-flash-lite on June 1. After today, API calls using the old model names return errors. Google's recommended path is a name swap to gemini-2.5-flash-lite, which carries identical $0.10 input and $0.40 output per-million pricing, so the migration is mostly a string change rather than a cost change. Teams still pinned to the 2.0 endpoints break today.

Source: Google Gemini deprecations, effective June 1, 2026

Why this matters before others see it: The story is not this one model, it is the cadence. Frontier labs are now sunsetting last-generation models on tight cycles, which turns "model EOL risk" into a real production concern rather than a footnote. A team that hardcoded a model name a year ago can wake up to 400 errors with little warning. The practical takeaway: pin to model aliases where the provider offers them, keep a migration runbook, and treat model lifecycle like dependency management. Expect more of these forced cutovers across every major API through 2026.
