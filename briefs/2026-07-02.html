Grid hits a record as DOE lets PJM push AI data centers onto backup power

Tier 1 High AI Data Centers

PJM Interconnection forecast 166,304 MW of electricity demand for July 2, above the grid's prior all-time record of 165,563 MW set in 2006. Two Department of Energy emergency orders under Section 202(c), signed June 30 and running through July 3, let PJM curtail data centers and other loads of at least 50 MW, requiring them to switch to onsite backup generators within 15 minutes of an emergency signal. PJM layered on a Maximum Generation Alert, a Load Management Alert, and a Low Voltage Alert, framing curtailment as a last resort before rolling blackouts. Source: utilitydive.com, datacenterdynamics.com

Why it matters: This is the first time the AI compute buildout collides with grid reliability at the level of a federal curtailment order, treating large data centers as an interruptible load the government can switch off to keep the lights on. The falsifiable test is repetition: if DOE issues another 202(c) curtailment authorization during a later summer or winter peak this year, interruptibility becomes a standing condition of siting AI campuses on the public grid rather than a one-off heat-wave measure. Watch whether operators respond by moving compute off-grid, because a rush to onsite generation is the signal that hyperscalers no longer trust the grid to carry their load.

GitHub Copilot moves browser control into the IDE at general availability

Tier 1 High AI Coding Tools

On July 1, GitHub made browser tools for Copilot in VS Code generally available and on by default. The agent can drive a real browser to open pages, click, type, read content, capture console errors, and take screenshots, then feed what it finds back into chat. User tabs stay private until Share with Agent is selected, and agent-opened tabs run isolated from personal cookies and storage, with an admin on/off switch plus site allow and deny lists. GitHub shipped it alongside two more July 1 general-availability moves: task-based automatic model routing in the Copilot CLI, and Copilot vision for reasoning over attached images and PDFs. Source: github.blog, github.blog

Why it matters: Giving a coding agent its own sandboxed browser closes the loop between writing code and checking it against a live page, which is the step that separates a code generator from something that can verify its own work. The falsifiable test is defaults following suit: if Cursor, Windsurf, or JetBrains ship an on-by-default in-IDE browser agent within two quarters, browser control becomes table stakes for coding assistants rather than a Copilot edge. Watch the admin allow and deny lists, because the moment enterprises start governing which sites an agent may drive is the moment agentic browsing is treated as production infrastructure, not a demo toy.
