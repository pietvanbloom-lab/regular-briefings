Tier 1 High confidence Early signal AI Coding

GitHub Copilot App enters technical preview as native desktop client with per-session work trees [source]

GitHub launched a standalone desktop app for Windows, macOS, and Linux that runs agentic development sessions tied to GitHub work. Each session gets its own git work tree, branch, and task state, enabling parallel agents on the same repo without conflicts. Available first to Copilot Pro and Pro+ subscribers; Business and Enterprise rollout through the week.

Why this matters before others see it: Per-session git work trees solve the single biggest correctness failure in parallel coding agents (clobbered branches). If GitHub gets this right at the OS layer, Cursor's "background agents" and Claude Code's `claude agents` are both pushed into a feature-parity race within 90 days. The Copilot Desktop App is also the first IDE-agnostic agent surface from GitHub, which signals long-term decoupling from VS Code.
