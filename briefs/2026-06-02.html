Z.ai releases GLM-5.1, a 754B-parameter open-weights MoE that tops open models on SWE-Bench Pro at 58.4

Tier 1 High confidence Early signal Open Source Coding

Why this matters before others see it: An MIT-licensed model that posts a SWE-Bench Pro number in the high 50s narrows the gap between open weights and the closed coding leaders to a margin that procurement teams will notice. The strategic angle is that self-hostable agentic coding is now a credible alternative just as the closed tools move to metered token billing, so the cost calculus flips for any team running heavy agent loops. Watch independent SWE-Bench Pro reproductions, whether inference providers list GLM-5.1 at a sharp price, and whether coding-tool vendors quietly add it as a backend. If the benchmark holds under third-party testing, this is the open-weight release that makes "host your own coding agent" a board-level option.

NVIDIA open-sources a collection of agent skills for physical AI that plug into Claude Code, Codex, and Cursor

Tier 1 High confidence Early signal Agentic AI Interop

NVIDIA released an open-source collection of Agent Toolkit tools and skills that let coding agents build, test, and run robotics, autonomous-vehicle, and vision-AI workflows as repeatable instructions. The skills integrate with Claude Code, OpenAI Codex, and Cursor through the open agentskills.io spec, with OSMO orchestration and robotics partners including 1X, Agility, NEURA, and Universal Robots. Code is published on GitHub.

Source: NVIDIA newsroom / GitHub (NVIDIA/skills), May 31, 2026

Why this matters before others see it: The headline is robotics, but the real signal is interop. A major hardware vendor shipping skills against a vendor-neutral agentskills.io spec that already runs on Claude Code, Codex, and Cursor is a bet that "skills" become portable across agents the way containers became portable across clouds. The second-order effect is that the moat shifts from the agent runtime to the skill library, which pressures every coding-agent vendor to either adopt the open spec or risk fragmentation. Watch whether the agentskills.io format gets more first-party backers, how it relates to MCP as the transport layer, and whether physical-AI skills become the wedge that pulls coding agents into the robotics stack.
