OpenAI ships ChatGPT Work, a multi-hour work agent, and folds Codex into a new ChatGPT desktop app

Tier 1HighAgentic AI

In a July 9 livestream OpenAI launched ChatGPT Work, a GPT-5.6-powered agent that takes an outcome, plans steps across connected apps and files, and stays on complex projects for hours, delivering finished documents, spreadsheets, presentations, and web apps. Why it matters: This is OpenAI productizing long-running agentic work, aimed squarely at the same territory as Claude Cowork, and the Codex-into-ChatGPT merge consolidates its coding and work surfaces into one app. Watch whether Plus and Business get it on schedule, what enterprise seats it wins in the next 30 days, and how Anthropic answers on the work-agent front.

Meta enters the paid API market: Muse Spark 1.1 opens to developers at $1.25 and $4.25 per million tokens

Tier 1HighEarly signalFrontier LLM

On July 9 Meta released Muse Spark 1.1 and, for the first time, opened a frontier Meta model to outside developers through the new Meta Model API, in public preview for US developers with $20 in free credits per account. Claimed specs: 1M-token context, computer use, multi-agent workflows, and native multimodal perception, with early partners including Replit, Cline, and Box. Pricing sits between the cheap tiers and mid tiers of OpenAI and Anthropic. Source: ai.meta.com

Why it matters: Meta abandoning its consumer-only posture means five frontier-class vendors now compete for the same enterprise API spend, and it lands the same day the GPT-5.6 and Grok 4.5 price points became public. Watch whether the preview pricing holds at GA, which routing platforms add Muse Spark first, and whether open-weight Llama releases slow now that Meta sells access directly.

Anthropic launches Reflect, a usage and wellbeing dashboard for Claude, in beta on Free, Pro, and Max

Tier 1HighEarly signalAnthropic

Anthropic introduced Reflect on July 9: a dashboard in Settings on Claude web and Desktop with a monthly recap of topics and usage patterns, quiet hours, break reminders, and skill suggestions built on the 4D AI Fluency Framework. It requires Memory on, excludes health-related chats entirely, and was built with input from MIT Media Lab AHA, Boston Children's Hospital Digital Wellness Lab, and FOSI. A Cowork reflection view is listed as available soon. Source: anthropic.com, techcrunch.com

Why it matters: Reflect is a wellbeing-framed engagement layer built on Memory, and TechCrunch already reads it as a retention play arriving amid broader AI-backlash coverage. Watch whether the Cowork reflection ships, whether usage dashboards become a category the other labs copy, and whether the wellbeing framing holds up once the monthly recaps start landing.

China's anthropomorphic AI rules bite: Alibaba disables Qwen humanlike agents today

Alibaba's July 6 notice takes effect today, July 10: Qwen's humanlike interactive agents and user-created agents are disabled, with broader agent services going offline July 15 when the Interim Measures on AI Anthropomorphic Interaction Services take force. The rules target sustained emotional-companion bots while sparing productivity agents; ByteDance's Doubao is following the same path and users are protesting lost chat histories. Source: scmp.com

A new beta of the v2 TypeScript SDK landed July 9 at 13:39 UTC targeting the 2026-07-28 protocol revision with its stateless core. Beta.3 adds Standard Schema support so elicitation accepts Zod objects, with pre-send rejection of shapes the restricted form schema cannot express, published across the server, node, and hono packages. The largest MCP revision since launch is now 18 days out with SDK betas in flight. Source: github.com

OpenAI Agents SDK v0.18.1 makes GPT-5.6 the default model

Published July 9 at 23:38 UTC, the Python Agents SDK switched default models and examples to GPT-5.6, with fixes for cache-write usage accounting, realtime session cleanup, and nested tool state restoration. Useful confirmation that GPT-5.6 is live in the agent stack the same day it launched, not just announced. LangGraph also shipped a minor 1.2.9 patch. Source: github.com
