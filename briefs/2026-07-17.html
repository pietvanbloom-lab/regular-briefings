Moonshot AI releases Kimi K3, the largest open-weight model on record at about 2.8 trillion parameters

Moonshot AI announced Kimi K3 on July 16, a Mixture-of-Experts model with roughly 2.8 trillion total parameters activating 16 of 896 experts, a 1M-token context window, and native multimodality built on new Kimi Delta Attention and Attention Residuals techniques. In blind Arena testing developers preferred K3 over leading US models for front-end coding, including Anthropic Fable 5 and OpenAI GPT-5.6, and it topped the Frontend Code Arena. One precise caveat: the model is API-only at launch, with full open weights and the technical report promised for July 27. Source: siliconangle.com

Why it matters: A Chinese lab is now setting the open-weight frontier on scale and, on early coding evals, on quality. Watch whether the weights actually land on July 27 as promised, and whether US labs respond on price, since K3 API rates undercut frontier closed models at $3 in and $15 out per million tokens.

Anthropic / Claude

1Password ships a Claude integration that lets the model act on logged-in sites without seeing credentials

1Password launched 1Password for Claude on July 16, a browser integration that lets Claude complete web tasks using a person's stored logins while credentials, including any one-time codes, are injected into the target site through a secure channel and never reach the model or Anthropic's systems. It is a concrete answer to the agent-security question of how autonomous agents authenticate without exposure. Source: siliconangle.com

Ledger launches Agent Stack, an open-source toolkit that gates agent crypto actions behind hardware approval

Ledger released Agent Stack on July 16, an open-source toolkit that lets AI agents read balances, analyze portfolios, and prepare transactions while every sensitive action must be physically approved on a Ledger hardware device, following an agents-propose, humans-approve model.
